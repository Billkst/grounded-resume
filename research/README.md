# research/

本目录存放 grounded-resume 项目所有研究阶段产出，包括 Baseline 评测数据、报告、模板及岗位分析资料。

## 目录结构

```
research/
├── baseline/           # Baseline 阶段全部评测数据与报告
│   ├── test-cases/     # 测试样本（Smoke / Deep）
│   ├── smoke-test/     # Smoke Test 输出与评分卡
│   ├── deep-test/      # Deep Test 输出与评分卡
│   ├── repo-reviews/   # D 类开源项目评审
│   ├── capability-matrix/  # 能力矩阵
│   └── reports/        # 综合分析报告
├── jd-analysis/        # 目标岗位 JD 分析资料
└── templates/          # 评测模板与样本模板
```
