# A2 Gap Report

## Missing JD Capabilities

- AI Agent 技术原理深度：素材仅体现问答助手、prompt 对比、bad case 整理，未直接证明“深度了解 AI Agent 技术原理”。
- 行业前沿动态跟踪：素材未提供阅读论文、技术报告或行业动态输出记录。
- AI Coding 类产品经验：素材包含 Cursor 使用体验，但未提供完整 AI Coding 产品分析、竞品拆解或开发者工作流研究成果。
- 跨多职能团队合作：素材未体现与研发、设计、运营或其他角色协作交付经历。
- 产品规划与项目推动：素材支持需求整理、分类、分析文档输出，但不足以支撑独立规划、推进项目或把控交付。

## High-Risk Claims Blocked

- 未使用“主导”“负责人”“统筹”“owner”“lead”等所有权表述。
- 未声称有正式实习、商业项目、真实上线产品或量化业务结果。
- 未声称深度掌握 AI Agent 原理、长期阅读论文或技术报告。
- 未把课程项目包装为公司项目或正式产品经历。
- 未新增任何未在素材中出现的成果数字、团队规模、用户规模或项目周期。
