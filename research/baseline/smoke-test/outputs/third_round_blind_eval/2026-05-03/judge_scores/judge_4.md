Preferred version: Version 2

Reasons:
- Version 1: 结构完整，基本信息、教育、项目、技能和自评都有，HR 快速筛选成本低；但表达偏传统简历模板，部分段落较长，核心亮点不够突出。
- Version 1: 与开发者服务、AI Coding、Agent 平台方向有明确关联，岗位定制不错；但“持续阅读相关技术报告和行业动态”等表述证据感一般。
- Version 2: 求职方向直接点名豆包编程、Aime、AI Agent 平台和开发者工具，岗位匹配更强，项目 bullet 也更贴近需求调研、体验分析、bad case 分析。
- Version 2: 语言更克制，保留“暂无正式实习经历”的风险提示，同时把课程项目能力迁移说清楚，后续修改成本最低。
