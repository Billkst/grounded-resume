- **主导构建** AI Native 文档问答系统，基于 LangChain 整合文档解析、向量索引（FAISS）与 OpenAI API 生成链路，支持 PDF/Markdown 多格式技术文档的语义化交互，覆盖 10+ 份文档、单篇最长 50+ 页
- **设计**结构化 Prompt 模板约束输出格式与引用规范，系统性执行 10+ 组端到端测试用例，定位 3 类引用偏差根因（上下文割裂、相似语义干扰、长文档位置偏移）
- **分析**检索-生成链路瓶颈，设计重排序（Re-rank）优化与上下文压缩策略，形成可复用的 RAG 调优方法论（分块策略、Embedding 选型、Prompt 工程）
- **基于** Dify 低代码平台快速验证 RAG 流程配置，对比分析平台级集成与框架级定制的灵活性差异，支撑后续 AI 研发工具链选型决策

### AI 任务规划与自主执行引擎 | 独立项目 | 2025.01 - 2025.04

- **设计并实现** AI 任务规划引擎，基于多轮提示词工程（Prompt Chain）将复杂指令拆解为可执行子任务序列，覆盖代码生成、数据处理、文档整理等 5+ 类研发辅助场景
- **构建** LLM 与外部工具的闭环调用机制（Function Calling），实现执行状态追踪、中间结果传递与异常分支处理，验证端到端自主执行流程可行性
- **建立**任务分解质量评估框架（步骤完整性、可执行性、依赖逻辑合理性），通过 5 轮次 Prompt 迭代与 Few-shot 示例优化，将步骤生成完整率提升 25+ 个百分点
- **调研** Multi-Agent 协作与 AI 自主交付模式，设计主控 Agent 调度 + 专项 Agent 执行的分层架构原型，探索任务并行化、结果聚合与冲突消解机制
